jQuery 的引入和入门：选择元素、$ & $() 下常用方法

2018-04-22 jQuery JavaScript, jQuery 字数统计: 798(字) 阅读时长: 3(分)

一、开始！

1. jQuery 的引入

jQuery官网、jQuery中文网、BootCDN

引入：

2. 加载问题

  1. $(function(){ })：等DOM加载完就可以执行了 , 性能要好
  2. $(document).ready(function(){ }) ：作用同window.onload

二、jQuery 写法

1. 方法函数化

获取元素，绑定事件等所有行为都要加括号

2. 链式操作

3. 取值赋值合体

同一个方法，根据参数区分取值和赋值

  • $('#div1').html() / $('#div1').html('hello')
  • $('#div1').css('width') /$('#div1').css('width','200px')

注意 ：当一组元素的时候，取值是一组中的第一个，赋值是一组中的所有元素

4. jQuery与 JavaScript 关系

可以共存，不能混用

三、jQuery 选择元素

1. 模拟css选择器

  1. $(’#div1’)
  2. $(‘div’)
  3. $(’.box’)
  4. $(’.ul1 li’)
  5. $(’[title=hello]’)

2. 独有表达式选择

(1). 按索引选择

注意有冒号

  1. $(‘li:eq(i)’) ：第i个元素，i从0开始
  2. $('li:even’) ：奇数行
  3. $(‘li:odd’) ：偶数行

(2). 筛选

  1. $(el).filter()：筛选
  2. $(el).not() ：不选某一元素 (针对元素自身)

​ 例 ：$('div').filter('.box').css('background','red');

(3). 包含、查找

  1. $(el).has() ：包含 (针对元素内部)
  2. $(el).find() ：内部寻找(可以通过模拟css选择器实现)

(4). 选择兄弟节点

  1. $(el).next()：上一个兄弟节点(不是所有)
  2. $(el).prev() ：下一个兄弟节点(不是所有)

四、$(el) 下常用方法

  1. $(el).attr() : 设置属性

    例 ：$('div').attr('class','box');

  2. $(el).html() ：获取元素中内容(同innerHTML)

  3. $(el).val() ：获取/修改元素value值

  4. $(el).text() ：获取一组元素中所有纯文本(不带标签)，把所有文字串联后返回(特例，获取所有)；设置时默认时纯文本(不会转成标签)

  5. $(el).index() ：索引，返回当前元素在所有兄弟节点中的位置

  6. $(el).addClass() / removeClass() ：添加 / 删除class

  7. $(el).size() ：获取一组元素的长度(类似length)，注意 $('#ui1 li').length 为jq下的属性

  8. $(el).each() ：jq中的循环

  9. $(el).get() ：把jq元素转成原生js，转成一个元素时必须要有参数(索引)，不加参数默认为一组元素。

五、$ 下的常用方法

  1. $.type() : 判断数据类型，但能判断的类型更精确

  2. $.trim() ：去掉字符串前后空格

  3. $.inArray(item, arr) : 类似于 indexOf，找不到返回-1

  4. $.proxy(function, newThis, [args…]) : 改变this指向，返回一个函数，

  5. $.noConflict() : 从全局放弃 $ 变量名，防止冲突。

    如 var wzr = $.noConflict() 则后续 wzr() 等同于 $()

  6. $.parseJSON() ：把严格模式的字符串转化成json对象

  7. $.makeArray() ：把类似数组对象(节点列表)转化成真正的js数组。

六、拓展插件

  1. $.extend() : 扩展工具方法下的插件形式
  2. $.fn.extend() : 扩展到jq对象下的插件形式，内部的this指向使用该方法的jq对象

用法 ：参数为一个json，key值为函数名，value为函数主体，可以包含多个函数
