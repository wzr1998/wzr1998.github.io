JavaScript 函数声明&函数表达式、作用域、异步&单线程

2017-11-26 JavaScript JavaScript 字数统计: 658(字) 阅读时长: 2(分)

一、JavaScript执行上下文

  1. 范围：一段<script></script>（全局）或者一段函数代码块

  2. 全局：var变量声明、函数声明会被提升

  3. 函数：var变量声明、函数声明、this、arguments会被提升

1. 函数声明

3. 函数声明与函数表达式区别：

  1. 函数表达式可以直接后面加括号执行，而函数声明是不可以的
  2. 函数声明会把整个代码块提升；函数声明上方可以直接调用
  3. 函数表达式会把函数名变量提升为undefeated；不能调用

三、作用域&作用域链

  1. this：要在执行时才能确认值，定义是无法确认
  2. 作用域：JS无块级作用域，只有全局作用域和函数作用域
  3. 作用域链：子作用域找不到变量到父作用域找，本作用域中没有的变量叫做自由变量
  4. 注意：函数作用域是在定义的时候确定的，不是在执行的时候确定

四、异步&单线程

  1. 什么是异步：异步不影响后续代码的执行，不会阻塞

  2. 什么时候需要异步：在所有可能发生等待的情况

  3. 使用异步的场景：

    1. 定时任务：setInterval、setTimeout
    2. 网络请求：AJAX请求、动态<img>加载（ img.onload = function(){} ）
    3. 时间绑定：不知道该事件什么时候发生

  5. setTimeout原理：

    1. 把setTimeout中函数放入暂存队列中
    2. 主线程中函数全部执行结束之后从缓存队列中取出第一个函数执行
    3. 注：setTimeout只能保证在指定的时间过后将任务(需要执行的函数)插入队列等候，并不保证这个任务在什么时候执行
