CSS3 transform 变换

2018-04-15 CSS CSS, CSS3 字数统计: 733(字) 阅读时长: 2(分)

CSStransform属性允许旋转，缩放，倾斜或平移给定元素。这是通过修改CSS视觉格式化模型的坐标空间来实现的。 链接 transform-function

一、transform 属性

  1. 作用：让元素在一个坐标系统中变形，这个属性包含一溪流变形函数，可以移动、旋转和缩放元素
  2. 语法：transform : none | <transform-function> [<transform-function>]...;
  3. 执行顺序：后写的先执行，先写的后执行，后执行的变换可能影响先执行的变换

二、transform-origin 坐标系统(变换基点)

transform-origin CSS属性可更改一个元素变形的原点，转换起点是应用转换的点。例如，rotate()函数的转换原点是旋转中心。（这个属性的应用原理是先用这个属性的赋值转换该元素，进行变形，然后再用这个属性的值把元素转换回去） 链接

语法：参数为一个值、两个值、三个值均可

  • 一个值：必须是<length>，<percentage> 或 left, center, right, top, bottom关键字中的一个。
  • 两个值：分别定义 x、y 轴偏移量，参数形式与一个值时相同
  • 三个值：前两个值和只有两个值时的用法相同，第三个值必须是 <length>，代表Z轴偏移量。

三、2D变换

  1. 旋转：rotate() / ['rotet]/

    参数为指旋转角度<angle>，角度增大顺时针旋转(正值顺时针，负值逆时针)

  2. 移动：translate() / translateX() / translateY()

    参数为移动距离<translation-value>，单位为px | %，translate()只写一个参数默认X轴偏移

  3. 缩放：scale() / scaleX() / scaleY() /[skeɪl]/

    参数为缩放比例(无单位) <number>，默认从中心缩放，scale()只写一个参数默认等比缩放

  4. 扭曲：skew() / skewX() / skewY() /[skjuː]/倾斜变换

    参数为斜切角度<angle>，有效值为-90deg～90deg，skew()只写一个参数默认为X轴扭曲

    1. skewX()方向：正值逆时针，负值顺时针
    2. skewY()方向：正值顺时针，负值逆时针

  5. 矩阵：matrix(a, c, b, d, tx, ty) /['meɪtrɪks]/

四、3D变换

  1. 旋转：rotate3d(x, y, z, angle) / rotateX() / rotateY() / rotateZ()

    rotate3d()中前三个参数可以表示该方向的旋转程度，参数都不允许省略

  2. 移动：translate3d(x, y, z) / translateZ() 改变对象离观察者远近//translateZ()通常用于遮照

    translate3d()中三个参数都不能省略

  3. 缩放：scale3d() / scaleZ()

    scale3d()中三个参数都不能省略

  4. 矩阵：matrix3d(sx, n, n, n, n, sy, n, n, n, n, sz, n, n, n, n, 1)

五、其他属性

  1. transform-style : falt | preserve-3d：规定如何在 3D 空间中呈现被嵌套的元素
  2. perspective : number(px)：景深，指定观察者与[z=0]平面的距离，使具有三维位置变换的元素产生透视效果
  3. perspective-origin : x-axis y-axis：指定透视点的位置(方位词或%)
  4. backface-visibility ： visible | hidden：指定元素背面面对用户时是否可见
