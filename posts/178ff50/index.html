ES6 函数的拓展

2018-05-20 JavaScript ES6, JavaScript 字数统计: 506(字) 阅读时长: 1(分)

一、参数默认值与rest参数

1. 参数默认值

ES6 中函数参数中可直接设置默认值。(本质是临时在参数部分用let声明变量)

  1. 参数默认值可以与解构赋值一起使用
  2. 使用变量设置默认值时参数默认值是惰性求值的
  3. 使用参数默认值时注意传入参数的参数位置
  4. 参数设置默认值时，参数部分会形成一个单独的作用域，初始化结束后，这个作用域就会消失(参数部分是单独的作用于)
  5. 通常情况下函数参数默认值都在参数结尾，不会在中间（中间的参数不能省略）

2. rest参数(…arg)

用于获取函数的多余参数，返回一个数组，必须放在形参列表末尾

二、函数的length / name 属性

fn.length：表示函数预期传入的参数个数，是指第一个具有默认值之前的参数个数(最少参数)

fn.name：返回该函数的函数名，如果将一个有名函数赋值给一个变量，返回这个有名函数原本的名字

三、箭头函数

  1. 箭头函数中没有arguments、super、new.target，多余参数用rest参数(…arg)获取
  2. 箭头函数里面根本自己的this，而是引用外层环境中的this
  3. 箭头函数的this是静态的，始终指向定义时的对象，而原函数形式的this是动态改变的，指向调用函数的对象
  4. 箭头函数执行部分

this指向问题： 箭头函数中的this继承的是父执行环境上下文里面的this，简单对象(非函数)是没有执行上下文的，不用在函数外部保存_this

四、注意：

  1. 在严格模式下，fn.caller和fn.arguments两个变量禁止使用
  2. ES2017允许函数参数列表的最后一个参数有逗号，规则与数组和对象保持一致
