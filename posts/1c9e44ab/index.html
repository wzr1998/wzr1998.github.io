Vue 事件处理器与自定义组件

2018-09-12 Vue Vue 字数统计: 658(字) 阅读时长: 2(分)

一、事件处理器

1. 绑定方法

  • v-on:event=“fn”
  • @event=“fn”

2. 事件修饰符

(1). 事件相关

  1. .stop：阻止事件冒泡
  2. .prevent：阻止默认事件
  3. .capture：阻止事件捕获
  4. .self：只在对象自身事件触发(内部判断target事件源)
  5. .once：事件只出发一次

(2). 按键修饰符

  1. .num：num为键值，当按下该按键时触发
  2. .enter /.tab / .delete / .esc / .space：按下enter / tab / delete / esc / 空格时触发
  3. .up / .down /.left / .right：按下上/下/左/右时触发

3. 注意

  1. v-on:event方法绑定事件内参数为函数名，可以加括号传递参数
  2. 事件处理器中绑定的函数要在选项数据methods对象中定义
  3. 传递参数时需要使用事件对象，需要在传参时手从传入event参数，即Fn(event, obj)
  4. Vue中建议事件处理函数中只有纯粹的事件逻辑，而不去处理事件细节

二、自定义组件

1. 创建自定义组件

创建 childComponent.vue 子组件

2. 父组件引入子组件

  1. 导入：import child from ‘@/components/childComponent.vue’
  2. 注册：components : { child }
  3. 使用：<child></child> 通过标签方式使用

(2). $emit自定义事件

子组件函数中使用：this.$emit('customEvent', arg) ，customEvent为自定义事件的事件名，满足条件直接调用

父组件中：<component @customEvent ="fn"></component> ，把父组件中函数传入子组件

4. slot 插槽

父组件调用子组件时，通过slot插槽向子组件插入内容。父组件中默认子组件标签内元素全部加入到第一个slot元素中，可通过父组件中slot属性把元素加入到子组件中对应位置
