HTML5 音视频 API

2018-03-17 HTML HTML, HTML5, JavaScript 字数统计: 496(字) 阅读时长: 2(分)

一、嵌入媒体

(1). 容器：

  1. 视频：<audio></audio> 链接
  2. 音频：<vidio></vidio> 链接
  3. 指定多种视频来源：<source></source> 链接
  4. 字幕：<track></track> 链接

(2). 示例：

二、媒体相关属性

(1). 通用属性

  1. controls： 显示或隐藏用户控制界面
  2. autoplay：媒体是否自动播放
  3. loop：体是否循环播放
  4. currentTime：开始到播放现在所用的时间
  5. duration：媒体总时间(只读)
  6. volume： 0.0-1.0的音量相对值
  7. muted： 是否静音
  8. autobuffer： 开始的时候是否缓冲加载，autoplay的时候，忽略此属性
  9. paused： 媒体是否暂停(只读)
  10. ended：媒体是否播放完毕(只读)
  11. error：媒体发生错误的时候，返回错误代码 (只读)
  12. currentSrc：以字符串的形式返回媒体地址(只读)

(2). video特有属性

  1. poster：视频播放前的预览图片
  2. width、height：设置视频的尺寸
  3. videoWidth、 videoHeight：视频的实际尺寸(只读)

三、媒体相关方法

  1. play()：媒体播放
  2. pause()：媒体暂停
  3. load()：重新加载媒体

四、媒体相关事件

参考链接

  1. ended：当目前的播放列表已结束时
  2. error：当在音频/视频加载期间发生错误时
  3. pause：当音频/视频已暂停时
  4. play：当音频/视频已开始或不再暂停时
  5. playing：当音频/视频在已因缓冲而暂停或停止后已就绪时
  6. seeking：当用户开始移动/跳跃到音频/视频中的新位置时
  7. seeked：当用户已移动/跳跃到音频/视频中的新位置时
  8. timeupdate：当目前的播放位置已更改时
  9. volumechange：当音量已更改时
