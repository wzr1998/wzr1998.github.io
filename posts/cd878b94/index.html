JavaScript 正则表达式

2018-06-18 JavaScript JavaScript, 正则表达式 字数统计: 931(字) 阅读时长: 3(分)

一、基本概念

1. 正则表达式

让计算机能够读懂人类的规则，用于操作字符串，默认区分大小写、匹配成功就会结束，不会继续匹配

2. JavaScript正则表达式写法

3. 单个字符、定位符

  1. | ：或
  2. 小括号() ：分组操作、匹配子项
  3. 字符类[] : 一组相似的元素，中括号的整体代表一个字符
  4. 排除^ ：如果^写在[]里面的话，就代表排除的意思
  5. 范围 ：[a-z]、[0-9]
  6. ^ : 正则的最开始位置，就代表起始的意思
  7. $ : 正则的最后位置 , 就代表结束的意思 /^\s+|\s+\$/g

二、转义字符

  1. 点 . ：任意字符（\. : 真正的点）
  2. \s、\S : 空格、非空格
  3. \d、\D : 数字、非数字
  4. \w、\W : 字符 ( 字母 ，数字，下划线_ )、非字符
  5. \b、\B : 独立的部分 （ 起始，结束，空格 ）、非独立的部分
  6. \1、\2… ：重复的第一(二)个子项

三、标识符

  1. i ：不区分大小写
  2. g：全局匹配
  3. m：多行匹配

四、量词

匹配字符不确定长度

  1. {a,b} : 最少出现a次，最多出现b次
  2. {a,} : 最少出现a次
  3. {a} : 正好出现a次
  4. +: {1,}、至少出现一次
  5. ? : {0,1}、出现0次或者1次
  6. *: {0,}、至少出现0次

五、方法

1. reg.test(str)

正则去匹配字符串，如果匹配成功就返回真，如果匹配失败就返回假

2. str.search(reg)

正则去匹配字符串 , 如果匹配成功，就返回匹配成功的位置，如果匹配失败就返回 -1

3. str.indexOf(str)

根据字符串查找字符串，如果匹配成功，返回匹配的第一个元素下标，匹配失败返回-1

3. str.match(reg)

正则去匹配字符串，如果匹配成功，就返回匹配成功的数组，如果匹配不成功，就返回null

4. str.replace(reg/newStr, replacement)

正则/字符串去匹配原字符串，匹配成功的字符去替换成新的字符串

replacement可以为新字符串或返回新字符串的函数，函数的第一个参数$0是匹配成功的字符(母项)，之后的$1, $2依次为第一子项、第二子项
