一、vue-router

1. 安装：

  • vue-cli2.0：npm install vue-router
  • vue-cli3.0：npm add router

2. router-link 标签跳转

(1). 跳转页面

其中to后写router-index.js中配置的路由，to后跟的就是url域名下实际的路径

(2). 跳转时传递参数

  • 其中name为路由js中命名的name值，params和query为传递的参数，后边接一个对象
  • 设置params传递参数后，路由文件需对应配置：path: '/demo2/:userName’，其中”:”代表不确定，”:”后边跟传递params中传递的参数名
  • 传递的params与路由中配置的params必须对应

3. 编程式导航跳转

更改 this.$router 对象实现路由调转

  1. 普通跳转： this.$router.push({ path: ‘/demo1’})
  2. 传递参数： this.$router.push({ name: ‘demo9’, params: { userId: 123 }, query: { plan: ‘private’ } })

4. 被跳转页面接收参数

  1. 每个模块this.$route下保存自身的路由信息
  2. this.$route.params 和 this.$route.query 下保存原页面传递的参数对象
